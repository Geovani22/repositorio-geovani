Desarrollo Web

Acerca del Curso

En este curso, aprenderá cómo crear su sitio web comenzando con un prototipo en papel. Se sumergirá en las mejores prácticas de desarrollo web al trabajar con HTML y CSS. Aprenderá sobre las herramientas para optimizar al máximo el sitio web e implementará el control de versiones del código mediante GIT y preprocesadores como SASS. Finalmente, aprenderá cómo aplicar Bootstrap a su proyecto y comprenderá la importancia del SEO en su desarrollo. Subirá su sitio web al servidor y aprenderá a utilizar el servicio. También sabrás presupuestar y cuidar a tus clientes finales.

¿A quién va dirigido?

Técnicos, tecnólogos en informática o campos afines, desarrolladores e ingenieros de sistemas o campos afines, analistas, consultores, consultores tecnológicos o web, integrantes del campo tecnológico, sistemas, desarrolladores, programadores, ingenieros o cualquier persona que quiera adquirir o mejorar conocimientos básicos de diseño web.

¿Qué aprenderás?

  • 👉 Construir Sitios y aplicaciones
  • 👉 Trabajar como Desarrollador Web Junior
  • 👉 Desarrollar aplicaciones móviles basadas en HTML
  • 👉 Desarrollar sitios webs con confianza
